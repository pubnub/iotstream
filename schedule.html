IoT Hands On: Build a Realtime App with Raspberry Pi 2 and PubNub
Class - Location TBD
Walk out of this workshop with your own IoT application, and a Raspberry Pi 2! Join Jasdeep, Tomomi and Bhavana as they walk you through building a realtime Internet of Things application using sensor data from Raspberry Pi 2. Just released, this hardware makes it quick and easy to build and deploy internet-enabled sensors. You'll also learn how to aggregate that sensor data and display it with realtime updates over the PubNub network.
